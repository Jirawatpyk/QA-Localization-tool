# Interactive Mockups

Interactive HTML mockups created to communicate design intent to the development team. Open in any browser.

| Mockup | File | Coverage |
|--------|------|----------|
| **Review View** | `ux-mockup-review-view.html` | Core action loop, Finding list, expanded FindingCard, Language Bridge, Detail Panel, Score Impact |
| **Dashboard** | `ux-mockup-dashboard.html` | Stats overview, Batch Summary (pass/review split), Upload zone, Recent Activity |
| **Upload Processing** | `ux-mockup-upload-processing.html` | 3-Layer Pipeline real-time progress, file cards per state (complete/processing/queued) |
| **QA Certificate** | `ux-mockup-qa-certificate.html` | Professional certificate document, score breakdown, findings summary, digital verification |
| **Command Palette** | `ux-mockup-command-palette.html` | Ctrl+K overlay, 3-tier search (Actions/Files/Findings), keyboard navigation |
| **PM Dashboard** | `ux-mockup-pm-dashboard.html` | Team performance, Reviewer assignment, AI Cost estimation, Quality trends |

All mockups located in: `_bmad-output/planning-artifacts/ux-mockup-*.html`
